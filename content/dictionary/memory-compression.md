Memory compression is the process of reducing the size of stored memories—conversation histories, task logs, or retrieved contexts—while preserving the most important information. Techniques include abstractive summarization (LLM-generated summaries), extractive compression (removing low-importance sentences), and hierarchical compression (summaries of summaries). Memory compression is essential for long-running agents where cumulative memory would otherwise exceed context limits or storage budgets.
