A multi-agent system comprises multiple AI agents that collaborate, communicate, and divide labour to accomplish tasks that exceed the capability or context window of a single agent. Agents can take specialised roles — a planner, a researcher, a critic, a coder — and pass results between each other in a structured workflow. Communication patterns include sequential handoffs (one agent's output feeds the next), parallel execution (multiple agents work simultaneously on subtasks), and debate (agents critique each other's outputs to improve quality). Challenges include coordination overhead, error propagation across agent boundaries, and ensuring the system as a whole remains aligned with the original user intent.
