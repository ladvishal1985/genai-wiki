A golden dataset is a curated set of high-quality (input, expected output) pairs used as the ground truth for evaluating an LLM application. It represents the "oracle" against which model outputs are compared. Building a good golden dataset requires diverse, representative examples covering edge cases, and regular updates as requirements change. The golden dataset is the foundation of regression testing and offline evaluation—its quality directly determines the reliability of evaluation signals.
