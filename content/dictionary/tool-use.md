Tool use (also called function calling) is the capability of a language model to decide to invoke external functions or APIs and incorporate their results into its response. The model is provided a schema of available tools — name, description, and parameter types — and generates a structured call when it determines a tool is needed rather than answering from parametric memory alone. Common tools include web search, code interpreters, calculators, database queries, calendar access, and custom business APIs. Tool use transforms LLMs from passive text generators into active agents capable of fetching real-time data, performing computations, and taking actions with side effects.
