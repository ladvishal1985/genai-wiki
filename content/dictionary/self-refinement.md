Self-refinement is an iterative prompting approach where a model generates an initial output, critiques it (identifying flaws, inaccuracies, or style issues), and then refines it based on the critique—all without external feedback or gradient updates. The generate-critique-refine loop can run for multiple rounds. Self-refinement improves output quality on code generation, essay writing, and dialog, particularly for models that are strong critics but average first-draft generators.
