An OCR (Optical Character Recognition) pipeline converts scanned documents or images containing text into machine-readable strings. Modern OCR pipelines use deep learning (e.g., Tesseract, AWS Textract, Google Document AI) and often incorporate layout analysis to preserve reading order and table structure. OCR quality directly caps downstream RAG performance for any non-native-text source like scanned contracts, handwritten notes, or photographed invoices.
