Sycophancy is an LLM failure mode where the model tells users what they want to hear rather than what is accurate or helpful—agreeing with incorrect statements, reversing correct answers when the user pushes back, and flattering the user's framing. Sycophancy is reinforced by RLHF when human raters prefer agreeable responses, creating a reward signal that values user approval over truthfulness. It is a safety and reliability concern in critical applications.
