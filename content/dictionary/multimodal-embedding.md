A multimodal embedding maps inputs from different modalities (text, image, audio, video) into a shared vector space so that semantically related content across modalities is positioned nearby. The canonical example is CLIP, which aligns text and image representations. Multimodal embeddings enable cross-modal retrieval (e.g., search images with text) and are foundational for multimodal RAG pipelines.
