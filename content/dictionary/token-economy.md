Token economy refers to the cost structure and management of LLM usage where pricing is per-token (input tokens + output tokens). Understanding the token economy means tracking cost per query, total monthly spend, cost per feature, and the ROI of optimizations like prompt compression, caching, and model routing. Engineering decisions—prompt length, context window size, sampling temperature (which affects output length), model tier selection—all have direct token economy implications.
