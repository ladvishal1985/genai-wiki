Structured memory stores agent knowledge in relational or document databases with explicit schemas rather than as free-form text vectors. Examples: a user profile table (name, preferences, history), a knowledge graph of entities and relationships, or a task queue. Structured memory enables precise, deterministic lookups (find user by ID) alongside semantic search for less-structured content. Hybrid memory systems combine vector memory (semantic recall) with structured memory (exact lookups).
