Episodic memory in LLM agents stores records of past experiences—specific interactions, tasks, tool call results, and outcomes—indexed by recency or similarity. When similar situations arise, relevant episodes are retrieved and provided as context, enabling the agent to learn from past mistakes or repeat successful strategies. Episodic memory is the foundation of Reflexion-style agents that improve over multiple task episodes without gradient updates.
