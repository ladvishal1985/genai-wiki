Metadata enrichment augments raw chunks with structured attributes—source URL, document title, author, publication date, section heading, page number, entity tags—that enable downstream filtering, attribution, and ranking. Enrichment can be rule-based (extracting from file paths), heuristic (heading detection), or LLM-generated (auto-tagging topics or entities). Rich metadata transforms a flat vector index into a queryable knowledge store.
