Model Context Protocol (MCP) is an open standard by Anthropic for connecting LLMs to external tools, data sources, and services through a unified protocol. MCP defines how servers expose tools (callable functions), resources (data), and prompts (templates) to LLM clients. By standardizing the interface, MCP allows LLMs to interact with any MCP-compliant service without custom integration code, analogous to how HTTP standardized web communication. MCP is gaining adoption as the connector layer for agentic AI applications.
