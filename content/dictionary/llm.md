A Large Language Model (LLM) is a transformer-based neural network trained on massive text corpora to predict and generate human-like text. Trained on hundreds of billions to trillions of tokens, LLMs develop broad linguistic capabilities and emergent skills — reasoning, translation, summarisation, and code generation — without task-specific training. Scale is a key driver: both model parameters and training data volume contribute to improved performance, as described by neural scaling laws. Examples include GPT-4, Claude, Gemini, LLaMA, and Mistral, spanning both closed-source API services and open-weight models.
